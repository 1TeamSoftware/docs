# 1TeamSoftware Documentation

Product documentation for WordPress plugins by [1TeamSoftware](https://1teamsoftware.com/).

## Plugins

- **[WP CloudSync Master](plugins/wp-cloudsync-master/README.md)** -- Offload WordPress media to cloud storage (Amazon S3, Cloudflare R2, Google Cloud, and more). Covers setup, provider configuration, integrations, and WP-CLI commands.

## Structure

```
plugins/
  wp-cloudsync-master/
    01-getting-started/    # Installation, overview, setup wizard
    02-cloud-providers/    # S3, R2, GCS connection guides
    03-core-configuration/ # Settings, sync queue, objects browser
    04-features-and-tools/ # Media scanner, cross-provider migration
    05-integrations/       # WooCommerce, image optimizers, cache plugins
    06-troubleshooting/    # Common issues, FAQ
    07-developer-guide/    # WP-CLI commands, action/filter hooks
```
